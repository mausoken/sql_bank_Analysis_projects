# Monthly Transaction Analysis

## Project Overview
This project analyzes monthly transaction trends to understand transaction volume and fee revenue over time.

## Dataset
Transaction data containing dates, amounts, fees, and transaction types.

## Tools Used
- PostgreSQL
- SQL

## Key SQL Techniques
- DATE_TRUNC for monthly grouping
- Aggregations (SUM, COUNT, AVG)
- ORDER BY

## Key Insights
- Monthly transaction volume trends were identified
- Fee revenue changes across months
- Average transaction amounts remain relatively stable
