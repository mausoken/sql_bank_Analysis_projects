# Sender and Receiver Analysis

## Project Overview
This project focuses on analyzing transaction flows by examining sender and receiver behavior.

## Dataset
Single table containing transaction ID, sender, receiver, amount, fee, date, and transaction type.

## Tools Used
- PostgreSQL
- SQL

## Key SQL Techniques
- GROUP BY
- Aggregations (SUM, COUNT)
- ORDER BY

## Key Insights
- A small number of senders generate most transactions
- Certain receivers account for higher total amounts
- Fee contribution varies by transaction type
