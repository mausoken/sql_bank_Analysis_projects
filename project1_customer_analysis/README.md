# Customer Transaction Analysis

## Project Overview
This project analyzes customer and transaction data using PostgreSQL to understand customer behavior and transaction activity.

## Dataset
The dataset contains customer information, account details, and transaction records

## Tools Used
- PostgreSQL
- SQL
- Excel

## Key SQL Techniques
- Table design and normalization
- PRIMARY KEY and FOREIGN KEY
- JOINs
- Aggregations (SUM, COUNT)

## Key Insights
- Identified customers with the highest transaction volume
- Analyzed total transaction amounts and fees per customer
- Observed transaction activity patterns
